# Contributing to The Great Riau: Math-Biz Mania

Thank you for your interest in contributing to The Great Riau: Math-Biz Mania.

We welcome contributions that improve the educational experience, gameplay mechanics, mathematical challenges, documentation, accessibility, and overall quality of the project.

## Getting Started

1. Fork this repository.
2. Clone your fork locally.

```bash
git clone https://github.com/your-username/great-riau-mathbiz.git
```

3. Install project dependencies.

```bash
npm install
```

4. Start the development server.

```bash
npm run dev
```

## Development Guidelines

Please follow these guidelines when contributing:

* Write clear and maintainable code.
* Follow the existing project structure and coding style.
* Test your changes before submitting.
* Keep pull requests focused on a single feature or fix.
* Update documentation when necessary.

## Reporting Issues

If you find a bug or have a suggestion:

1. Check existing Issues first.
2. Create a new Issue if one does not already exist.
3. Provide a clear description of the problem or enhancement.

## Pull Request Process

1. Create a new branch for your changes.

```bash
git checkout -b feature/your-feature-name
```

2. Commit your work.

```bash
git commit -m "Add: short description of change"
```

3. Push your branch.

```bash
git push origin feature/your-feature-name
```

4. Open a Pull Request and describe your changes.

## Areas for Contribution

Contributors are encouraged to help with:

* Mathematics learning content
* Business simulation mechanics
* User interface improvements
* Accessibility enhancements
* Performance optimization
* Documentation improvements
* Backend and database integration (future development)
* Testing and bug fixing

## Code of Conduct

Please be respectful and constructive when interacting with other contributors.

## Thank You

Every contribution, whether it is code, documentation, testing, or feedback, helps improve The Great Riau: Math-Biz Mania and supports its mission of making mathematics and entrepreneurship learning more engaging and accessible.
