## [1.0.0] - 2026-06-03

### Added

#### Core Gameplay

* Educational business simulation gameplay.
* Progression system based on player decisions.
* XP and budget management mechanics.
* Multiple gameplay stages representing a complete business cycle.

#### Learning Modules

* Mathematical problem-solving challenges.
* Dynamic inequality exercises.
* Linear programming optimization activities.
* Interactive business decision-making scenarios.

#### Business Simulation

* Employee recruitment system.
* Initial capital selection system.
* Production management workflow.
* Batch production configuration.
* Customer interaction and sales simulation.

#### Data Persistence

* Progress storage using Local Storage.
* Session persistence across browser refreshes.
* Historical run tracking and player statistics.

#### Visualization

* SVG-based graph rendering.
* Mathematical coordinate visualization.
* Statistical review dashboard.
* End-of-session performance analysis.

#### User Interface

* Responsive web interface.
* Riau Malay-inspired visual identity.
* Multi-page navigation using React Router.
* Modern UI built with Tailwind CSS.

#### Technical Features

* Built with React 19 and Vite.
* Context API state management.
* Automatic deployment through Vercel.
* Public live demo availability.

### Documentation

* Added comprehensive README.
* Added MIT License.
* Added CONTRIBUTING guidelines.
* Added project screenshots.
* Added deployment information.
* Added future development roadmap.
